# 🍔 Swiggy Sales Analysis

This project analyzes Swiggy sales data to understand customer behavior, sales trends, and business insights using Python Data Analysis libraries.

## 📂 Dataset
- File: `swiggy_data.xlsx`
- Contains sales, orders, customers, and restaurant-related information

## 🚀 Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Plotly  

## 📊 Project Tasks
- Data Cleaning  
- Exploratory Data Analysis (EDA)  
- Data Visualization  
- Sales Trend Analysis  
- Business Insights
