# 🍔 Swiggy Sales Analysis

This project analyzes **Swiggy Sales Data** to understand customer behavior, sales trends, and business insights using Python Data Analysis and Visualization libraries.

---

# 📌 Project Overview

The goal of this project is to analyze Swiggy sales data and extract meaningful insights such as:

- Sales trends  
- Customer behavior  
- Popular restaurants  
- Top cities for orders  
- Revenue analysis  
- Order distribution  

This project uses **Exploratory Data Analysis (EDA)** and **Data Visualization** techniques.

---

# 📂 Dataset Information

**Dataset Name:** Swiggy Sales Dataset  

### Dataset Features

The dataset includes: 
 
- Order ID  
- Customer Name  
- City  
- Restaurant Name  
- Cuisine  
- Order Date  
- Delivery Time  
- Order Amount  
- Rating  
- Payment Method
